Pour utiliser cette application vous devez installer:
-Python
-PostgreSQL avec le mot de passe : postgres
L'option "First config" est très importante et doit s'ouvrir avant l'application car c'est qui effectue les premières configurations avant l'utilisation.
Pour cette opération veuillez à vous munir d'une connexion à Internet stable.
Elle se trouve dans le dossier de l'application.
